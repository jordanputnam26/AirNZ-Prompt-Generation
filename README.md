# Air NZ Prompt Building Workspace

This repository contains the working conversation-design prompt-building workspace for Air New Zealand servicing prompts.

The main project lives in:

- `air-nz-conversation-design/`

That workspace contains:

- Air NZ foundation artefacts
- scenario briefs and test cases
- prompt drafts
- source-material notes

Current focus:

- `Claim Missing Airpoints`

Workflow:

1. Use workshop method and prompt patterns
2. Review source material
3. Create a small set of context artefacts
4. Build scenario prompts from those artefacts
5. Test and refine
